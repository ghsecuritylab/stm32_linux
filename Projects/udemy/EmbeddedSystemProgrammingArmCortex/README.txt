This directory contains the examples used in the lecture "Embedded system programming on arm cortex m3/m4" on Udemy
https://www.udemy.com/embedded-system-programming-on-arm-cortex-m3m4/learn/v4/overview
They are modified in order to run on stm32f429i discovery.

Original version (for stm32f407 discovery, provided by the instructor Kiran Nayak) 
can be found here : https://www.dropbox.com/s/9pqf204fm6yo0yo/Lab_src_code.rar?dl=0
